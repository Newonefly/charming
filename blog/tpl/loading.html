加载过渡

pace插件

文件轻量实用,仅12k左右,在事件循环滞后，会在页面记录加载的状态以及进度情况, 可以兼容IE8以上的所有主流插件官方提供了一些加载动画, 也可以自己手写.

调用方法：

在head标签中插入:

< link rel="stylesheet" href="/pace/themes/pace-theme-barber-shop.css"> // 可根据他给的类做任意修改

< script src="/pace/pace.min.js">

直接把文件引进去就ok了,它会自动检测页面进度,至于样式效果可以观看一下官方给的样式,自己进行修改

还可以通过设置window.paceOptions做一些自定义配置:

paceOption = {

// Disable the 'elements' source

elements: false,

// Only show the progress on regular and ajax-y page navigation

restartOnRequestAfter: false

}

或者直接写在script标签内

< script data-pace-options='{ "ajax": false }' src='pace.js'>

如果你使用AMD或者Browserify来加载模块的话，你可以通过这样子来设置（例如：start:

define(['pace'], function(pace){

pace.start({

document:false

});

});

附加一些API:

  • Pace.start：开始显示进度条，如果你不是使用AMD或者Browserify来加载模块的话，这个会默认执行.
  • Pace.restart：进度条重新加载以及显示
  • Pace.stop：隐藏进度条以及停止加载。
  • Pace.track：监测一个或者多个请求任务。
  • Pace.ignore：忽略一个或者多个请求任务。

css 绝对底部(Sticky Footer)

假设一个页面结构 :

< div class="wrapper">

< div class="content">页面主体内容区域 < /div>

< div class="footer">需要做到 Sticky Footer 效果的页脚 < /div>

< /div>

方法一 : absolute 绝对定位

html,body {

height:100%;

}

.wrapper {

position: relative;

min-height: 100%;

padding-bottom: 50px;

box-sizing: border-box;

}

.footer {

position: absolute;

bottom: 0;

height: 50px;

需要指定html、body 100%的高度,另外就是保证content的padding-bottom的height与footer一致

方法二: calc

通过计算函数 calc 计算（视窗高度 - 页脚高度）赋予内容区最小高度，不需要任何额外样式处理，代码量最少、最简单。

.content{ min-height:calc(100vh - 50px );}
.footer{ height:50px ;}

但是尴尬的事存在移动设备兼容性问题，不过没有几个兼容性问题那能叫前端吗，姑且留下

方法三: Flexbox

伸缩盒很好用，但是总觉得属于世外高人，自成一派,与江湖格格不入

html{ height: 100%; }
body{ min-height: 100%; display: flex; flex-direction: colum;}
.content{ flex: 1;}

flex写法: 神功 心法
